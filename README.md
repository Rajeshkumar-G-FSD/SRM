# SRM Monorepo

This workspace contains three apps:

- `frontend`: Vite + React app
- `next-app`: Next.js app
- `server`: Node.js Express backend

## Commands

Install dependencies:

```bash
npm install
```

Run the frontend:

```bash
npm run dev:frontend
```

Run the Next.js app:

```bash
npm run dev:next
```

Run the server:

```bash
npm run dev:server
```
